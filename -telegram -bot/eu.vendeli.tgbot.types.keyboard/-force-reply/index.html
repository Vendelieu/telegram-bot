ForceReply

@Serializable
data class ForceReply(val inputFieldPlaceholder: String? = null, val selective: Boolean? = null) : Keyboard

Upon receiving a message with this object, Telegram clients will display a reply interface to the user (act as if the user has selected the bot's message and tapped 'Reply'). This can be extremely useful if you want to create user-friendly step-by-step interfaces without having to sacrifice privacy mode. Api reference: https://core.telegram.org/bots/api#forcereply
